## 스파르타마켓 - 우리를 위한 중고 거래
#### 내일배움캠프 AI 6기 django 심화 개인 프로젝트

### 프로젝트 소개
중고 물품을 거래하는 플랫폼 웹 페이지 django DRF 백엔드 구현.


### 💻 개발 환경
- Python
- django
- SQLite

### 📂 주요 기능
-----
#### 회원 기능 (accounts)
- 회원가입
- 로그인
- (프로필 페이지)

#### 게시글 기능 (products)
- 물건 목록 조회
- 물건 상세 조회: 제목, 내용, 가격
- 글 작성, 수정, 삭제
